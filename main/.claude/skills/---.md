# Single-Variant Health Risk Markers

## Overview

Some genetic variants have well-established, individually significant effects on
disease risk. Unlike PGS (which aggregate thousands of small effects) or monogenic
screening (which looks for rare disease-causing mutations), these are common or
moderate-frequency variants where knowing the genotype directly informs health
risk or clinical management.

This pipeline:

1. Genotypes each marker from the sample VCF
2. Interprets the genotype using a variant-specific risk table
3. Contextualizes with population frequency and ancestry
4. Generates a report with per-marker risk assessment

### Genotyping types

| Type | Description | Example |
|------|-------------|---------|
| `single_snv` | One variant, interpret REF/ALT genotype directly | Factor V Leiden, MTHFR C677T |
| `haplotype` | Two or more variants define a haplotype | APOE (rs429358 + rs7412) |
| `presence` | Specific pathogenic variant — report present/absent | BRCA1 185delAG |
| `allele_count` | Count risk alleles, interpret as dosage | FTO rs9939609 (0/1/2 copies of A) |

## Pipeline Steps

### Step 1: Extract Genotypes at All Marker Positions

```bash
SAMPLE_VCF="$1"
MARKER_LIST="$2"  # JSON file
TMPDIR=$(mktemp -d)

# Build positions file from all markers
python3 << SCRIPT
import json
markers = json.load(open("$MARKER_LIST"))
with open("$TMPDIR/marker_positions.tsv", "w") as f:
    for m in markers:
        gt = m["genotyping"]
        if gt["type"] == "haplotype":
            for v in gt["variants"]:
                pos = v["grch38"]
                f.write(f"{pos['chr']}\t{pos['pos']}\n")
        elif gt["type"] in ("single_snv", "allele_count"):
            pos = gt["variant"]["grch38"]
            f.write(f"{pos['chr']}\t{pos['pos']}\n")
        elif gt["type"] == "presence":
            pos = gt["variant"]["grch38"]
```

### Step 2: Genotype Each Marker

```python
def genotype_markers(calls_file: str, marker_list: list) -> list:
    """
    Determine genotype for each marker and look up interpretation.
    """
    # Parse calls into lookup
    calls = {}
    with open(calls_file) as f:
        for line in f:
            parts = line.strip().split('\t')
            if len(parts) >= 5:
                chrom, pos = parts[0], int(parts[1])
                ref, alt, gt = parts[2], parts[3], parts[4]
                gq = int(parts[5]) if len(parts) > 5 and parts[5] != '.' else None
                dp = int(parts[6]) if len(parts) > 6 and parts[6] != '.' else None
                calls[(chrom, pos)] = {
                    "ref": ref, "alt": alt, "gt": gt, "gq": gq, "dp": dp
                }

    results = []
    for marker in marker_list:
        gt_config = marker["genotyping"]

        if gt_config["type"] == "single_snv":
            result = _genotype_single_snv(marker, gt_config["variant"], calls)
        elif gt_config["type"] == "allele_count":
            result = _genotype_single_snv(marker, gt_config["variant"], calls)
        elif gt_config["type"] == "haplotype":
            result = _genotype_haplotype(marker, gt_config, calls)
        elif gt_config["type"] == "presence":
            result = _genotype_presence(marker, gt_config["variant"], calls)
        else:
            result = {"error": f"Unknown genotyping type: {gt_config['type']}"}

        result["marker_name"] = marker["marker_name"]
        result["gene"] = marker["gene"]
        result["category"] = marker.get("category", "")
        result["clinical_notes"] = marker.get("clinical_notes", "")
        result["actionability"] = marker.get("actionability", "")
        results.append(result)

    return results


def _genotype_single_snv(marker, variant, calls):
    """Genotype a single-SNV marker."""
    pos = variant["grch38"]
    key = (pos["chr"], pos["pos"])
    call = calls.get(key)

    if call is None:
        # Position not in VCF — assume homozygous reference
        ref_allele = pos["ref"]
        genotype_str = f"{ref_allele}{ref_allele}"
        gt_raw = "0/0"
    else:
        gt_raw = call["gt"]
        ref_allele = pos["ref"]
        alt_allele = pos["alt"]

        if gt_raw in ("0/0", "0|0"):
            genotype_str = f"{ref_allele}{ref_allele}"
        elif gt_raw in ("0/1", "0|1", "1|0"):
            genotype_str = f"{ref_allele}{alt_allele}"
        elif gt_raw in ("1/1", "1|1"):
            genotype_str = f"{alt_allele}{alt_allele}"
        else:
            genotype_str = gt_raw

    # Look up interpretation
    interp = marker.get("interpretation", {}).get(genotype_str, {})
    if not interp:
        # Try reversed (e.g., "TC" vs "CT")
        rev = genotype_str[::-1]
        interp = marker.get("interpretation", {}).get(rev, {})

    return {
        "genotype": genotype_str,
        "gt_raw": gt_raw,
        "rsid": variant.get("name", ""),
        "risk_level": interp.get("risk_level", "unknown"),
        "effect": interp.get("effect", "No interpretation available"),
        "frequency": interp.get("frequency", ""),
        "gq": call.get("gq") if call else None,
        "dp": call.get("dp") if call else None,
    }


def _genotype_haplotype(marker, gt_config, calls):
    """Genotype a multi-variant haplotype (e.g., APOE)."""
    if gt_config.get("haplotype_logic") == "apoe_special":
        return _genotype_apoe(marker, gt_config["variants"], calls)

    # Generic haplotype: concatenate alleles from each variant
    alleles = []
    for v in gt_config["variants"]:
        pos = v["grch38"]
        key = (pos["chr"], pos["pos"])
        call = calls.get(key)
        if call:
            alleles.append({"gt": call["gt"], "ref": pos["ref"], "alt": pos["alt"]})
        else:
            alleles.append({"gt": "0/0", "ref": pos["ref"], "alt": pos["ref"]})

    # Build haplotype string (simplified — proper phasing requires read-backed phasing)
    hap_str = "/".join(a["gt"] for a in alleles)
    interp = marker.get("interpretation", {}).get(hap_str, {})

    return {
        "genotype": hap_str,
        "risk_level": interp.get("risk_level", "unknown"),
        "effect": interp.get("effect", "See interpretation table"),
        "frequency": interp.get("frequency", ""),
    }


def _genotype_apoe(marker, variants, calls):
    """
    Special APOE genotyping logic.

    APOE alleles defined by two SNPs:
      rs429358 (pos 44908684): T=e3/e2 site, C=e4 site
      rs7412   (pos 44908822): C=e3/e4 site, T=e2 site

    Haplotype definitions:
      e2: rs429358=T, rs7412=T
      e3: rs429358=T, rs7412=C  (reference)
      e4: rs429358=C, rs7412=C
    """
    # Find the two variants
    v1 = next(v for v in variants if v["name"] == "rs429358")
    v2 = next(v for v in variants if v["name"] == "rs7412")

    pos1 = v1["grch38"]
    pos2 = v2["grch38"]

    call1 = calls.get((pos1["chr"], pos1["pos"]))
    call2 = calls.get((pos2["chr"], pos2["pos"]))

    # Determine alleles at each site
    gt1 = call1["gt"] if call1 else "0/0"
    gt2 = call2["gt"] if call2 else "0/0"

    # rs429358: ref=T, alt=C. Count C alleles.
    c_count_1 = _count_alt(gt1)  # 0, 1, or 2 copies of C (e4 marker)
    # rs7412: ref=C, alt=T. Count T alleles.
    t_count_2 = _count_alt(gt2)  # 0, 1, or 2 copies of T (e2 marker)

    # Determine diplotype
    # This is unphased, so we infer the most likely diplotype:
    apoe_diplotype = _infer_apoe_diplotype(c_count_1, t_count_2)

    interp = marker.get("interpretation", {}).get(apoe_diplotype, {})

    return {
        "genotype": apoe_diplotype,
        "rs429358_gt": gt1,
        "rs7412_gt": gt2,
        "rsids": "rs429358 + rs7412",
        "risk_level": interp.get("risk_level", "unknown"),
        "effect": interp.get("effect", ""),
        "frequency": interp.get("frequency", ""),
        "gq": min(
            call1.get("gq", 99) if call1 else 99,
            call2.get("gq", 99) if call2 else 99,
        ),
    }


def _count_alt(gt: str) -> int:
    """Count ALT alleles in a GT field."""
    gt = gt.replace("|", "/")
    alleles = gt.split("/")
    return sum(1 for a in alleles if a == "1")


def _infer_apoe_diplotype(c_at_429358: int, t_at_7412: int) -> str:
    """
    Infer APOE diplotype from unphased genotype counts.

    rs429358 C alleles (e4 marker): 0, 1, or 2
    rs7412 T alleles (e2 marker): 0, 1, or 2

    Possible diplotypes:
      e2/e2: C=0, T=2
      e2/e3: C=0, T=1
      e3/e3: C=0, T=0
      e2/e4: C=1, T=1  (ambiguous without phasing — e2/e4 vs e1/e3, but e1 is extremely rare)
      e3/e4: C=1, T=0
      e4/e4: C=2, T=0
    """
    lookup = {
        (0, 0): "e3/e3",
        (0, 1): "e2/e3",
        (0, 2): "e2/e2",
        (1, 0): "e3/e4",
        (1, 1): "e2/e4",
        (2, 0): "e4/e4",
        (2, 1): "unknown",  # e1/e4? Extremely rare
        (1, 2): "unknown",
        (2, 2): "unknown",
    }
    return lookup.get((c_at_429358, t_at_7412), "unknown")


def _genotype_presence(marker, variant, calls):
    """Check presence/absence of a specific pathogenic variant."""
    pos = variant["grch38"]
    key = (pos["chr"], pos["pos"])
    call = calls.get(key)

    if call is None or call["gt"] in ("0/0", "0|0"):
        return {
            "genotype": "not_detected",
            "risk_level": "average",
            "effect": "Variant not detected",
            "rsid": variant.get("name", ""),
        }

    gt = call["gt"]
    if gt in ("0/1", "0|1", "1|0"):
        zygosity = "heterozygous"
    elif gt in ("1/1", "1|1"):
        zygosity = "homozygous"
    else:
        zygosity = gt

    return {
        "genotype": f"detected ({zygosity})",
        "zygosity": zygosity,
        "risk_level": "high",
        "effect": marker.get("interpretation", {}).get("detected", {}).get("effect", "Pathogenic variant detected"),
        "rsid": variant.get("name", ""),
        "gq": call.get("gq"),
        "dp": call.get("dp"),
    }
```

```
| PCSK9 R46L | PCSK9 | CT | 🔵 Protective | ~50% lower LDL |
| ... | | | | |
```

Risk level icons: 🔴 High, 🟡 Elevated, 🟢 Average, 🔵 Protective/Reduced.

**2. Per-Marker Detail Sections** with genotype, interpretation, population
frequency in context of sample ancestry, clinical notes, and actionability.

**3. Disclaimers:**

```
IMPORTANT:
- These markers represent well-studied genetic associations, not diagnoses.
- Risk estimates (e.g., "3x risk") are relative risks compared to the reference
  genotype, not absolute probabilities.
- Environmental, lifestyle, and other genetic factors substantially modify
  actual disease risk. A "high risk" genotype does not mean disease is inevitable.
- Population frequencies are approximate and may vary by specific sub-population.
- Some genotype interpretations assume unphased data. Haplotype markers (e.g., APOE)
  are inferred from unphased genotypes and may rarely be ambiguous.
- Clinical action should be discussed with a healthcare provider.
```

## Error Handling

| Error | Cause | Resolution |
|-------|-------|------------|
| Position not in VCF, no BAM available | Assume homozygous reference; flag as "inferred ref/ref — no direct observation" |
| Low GQ (<20) at marker position | Unreliable genotype call | Report with warning; recommend confirmation |
| APOE diplotype = "unknown" | Unusual allele combination (e.g., e1) | Report raw genotypes at both SNPs; note rarity |
| Multiallelic site at marker position | Multiple ALT alleles | Extract the specific ALT matching the marker; ignore others |
| Marker not in genome build | Build mismatch | Verify sample build; liftover coordinates if needed |
